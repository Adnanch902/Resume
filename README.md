# Adnan Azad Portfolio Resume

This repository contains the source code for Adnan Azad's interactive portfolio and resume website.

## Highlights

- Leadership, architecture, and engineering profile presentation
- Certifications and tech highlights sections
- Case-study style work carousel (challenge, role, impact)
- Resume download and Calendly meeting booking
- Responsive React + TypeScript UI with animation and 3D elements
